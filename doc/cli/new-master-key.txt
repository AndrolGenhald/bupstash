Archivist new-master-key: Create a new master archive key.

Description:
  Creates a new master key and writes it to disk with permissions
  that are readable only by the creating user.

  Keep this key private and secure as it is needed to read
  any archived data sent using client keys derived from the
  master key.

Quick examples:

  $ archivist new-master-key -o ./backups-master.key
