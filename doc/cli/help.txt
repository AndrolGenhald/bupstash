bupstash: Easy, Secure, Encrypted, Deduplicated Backups

Subcommands:

'init'             - Initialize a bupstash repository.
'new-key'          - Create a key used for reading and writing encrypted data.
'new-send-key'     - Create a send key used for writing encrypted data.
'new-metadata-key' - Create a metadata key used for searching and listing.
'help'             - Print this message.

For subcommand specific help, run 'bupstash SUBCMD --help'
